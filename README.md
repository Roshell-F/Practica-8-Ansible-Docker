En este proyecto se implementó un entorno de automatización utilizando Docker y Ansible. Primero, se crearon cinco contenedores de Ubuntu que simulan servidores, los cuales fueron levantados correctamente y verificados en ejecución. Luego, se estableció la conexión mediante SSH, configurando acceso sin contraseña a través de claves para permitir la comunicación automática. Posteriormente, se configuró Ansible mediante los archivos de inventario y configuración, logrando la conexión exitosa con todos los servidores, comprobada con el comando de prueba (ping). Finalmente, se ejecutó un playbook que automatizó diversas tareas como la actualización de paquetes, la creación de un usuario, la generación de directorios y archivos, y la instalación de aplicaciones. Los resultados muestran que todas las tareas fueron ejecutadas correctamente en cada servidor, cumpliendo con los objetivos de la práctica.
